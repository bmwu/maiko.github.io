Reward 图片说明

使用说明

请将以下图片放在 source/images/ 目录下：

  1. 微信支付二维码 - wechatpay.jpg

    • 建议尺寸：200x200 像素
    • 格式：JPG/PNG
    • 内容：你的微信收款二维码

  2. 支付宝二维码 - alipay.jpg

    • 建议尺寸：200x200 像素
    • 格式：JPG/PNG
    • 内容：你的支付宝收款二维码

生成二维码的方法

微信支付

  1. 打开微信
  2. 点击右上角的 “+”
  3. 选择”收付款”
  4. 点击”二维码收款”
  5. 点击”保存收款码”
  6. 将保存的图片重命名为 wechatpay.jpg 并放入 source/images/ 目录

支付宝

  1. 打开支付宝
  2. 点击首页的”收钱”
  3. 点击”保存图片”
  4. 将保存的图片重命名为 alipay.jpg 并放入 source/images/ 目录

注意事项

  • 图片文件名必须与配置文件中的路径保持一致
  • 建议图片大小控制在 100KB 以内，确保加载速度
  • 可以使用图片压缩工具优化文件大小

添加图片后，reward 功能就会正常显示在文章底部。
